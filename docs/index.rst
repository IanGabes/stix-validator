Welcome to stix-validator's documentation!
==========================================

The **stix-validator** library and scripts helps validate STIX content
using `XML Schema`_, `STIX Suggested Practices`_, and `STIX Profile`_ validation
scenarios.

.. _XML Schema: http://stix.mitre.org/language/
.. _STIX Profile: http://stixproject.github.io/documentation/profiles/
.. _STIX Suggested Practices: http://stixproject.github.io/documentation/suggested-practices/

For more information about STIX, please visit the
`STIX website <http://stix.mitre.org/>`_. For more information about CybOX,
please visit the `CybOX website <http://cybox.mitre.org>`_.

Contributing
============
If a bug is found, a feature is missing, or something just isn't behaving the
way you'd expect it to, please submit an issue to our `tracker`_. If you'd like
to contribute code to our repository, you can do so by issuing a `pull request`_
and we will work with you to try and integrate that code into our repository.

.. _tracker: https://github.com/STIXProject/stix-validator/issues
.. _pull request: https://help.github.com/articles/using-pull-requests
